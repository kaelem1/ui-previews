Clawdbot 记忆系统拆解

通过本地 Markdown + 向量检索的组合，让 AI 在对话中“记得住、找得到、用得上”。

本地持久化 MemorySearch + BM25 NotebookLM 风格总结

Summary

  • 记忆以 Markdown 文件落在工作区，分日记与长期记忆。
  • 对话收尾触发总结写入，形成可检索的时间线。
  • MemorySearch 负责检索，拼接进上下文。

Key Evidence

  • 启用索引后，检索不再逐文件扫描。
  • 索引构建有防抖阈值，避免频繁重建。
  • 混合检索：向量相似度 + BM25。

Takeaways

  • 记忆系统核心是 索引与召回。
  • 要“记得住”，需开启自动总结与长期记忆。
  • 要“找得到”，必须启用向量检索。
