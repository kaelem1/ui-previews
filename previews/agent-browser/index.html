Agent Browser

更省 Token 的浏览器自动化

通过把网页操作压缩成“可交互元素 + 短指令”，让模型专注完成任务而不是吞 DOM。

Vercel 开源 元素级控制 低成本调用

Summary

  • 传统自动化传输整页源码，浪费 token。
  • Playwright 抽象交互元素，已大幅减负。
  • Agent Browser 用短指令控制元素，进一步压缩输入。

Key Evidence

  • 元素列表替代 HTML/CSS/JS 细节。
  • 指令式操作（click E2）比代码驱动更轻量。
  • 更适合大模型“看得懂、用得上”。

Takeaways

  • 省 token 的关键是减少无关 DOM 噪音。
  • 自动化任务优先考虑 Agent Browser。
  • 复杂测试脚本仍可用 Playwright。
